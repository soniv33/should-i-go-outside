# Should I Go Outside?

One verdict. No dashboard. Just yes or no.

**Live demo:** `https://<your-username>.github.io/Shouldigooutside/`

---

## What it does

Checks your local weather (temperature, rain, UV, wind) and air quality (AQI via PM2.5) and returns a single verdict with a one-line reason. Nothing else.

Data sources: [Open-Meteo](https://open-meteo.com/) weather + air quality APIs — free, no API key required.

---

## Deploy to GitHub Pages

1. Fork this repo
2. Go to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose branch `main`, folder `/ (root)`, and save
5. Your site will be live at `https://<your-username>.github.io/<repo-name>/` within a minute

No build step. No dependencies. No backend.
